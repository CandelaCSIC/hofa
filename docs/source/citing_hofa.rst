Citing HoFa
======================================

If you would like to acknowledge the HoFa project in your academic publication, we suggest citing the following paper:

    Pablo Candela, Diego González-Sánchez, and Balázs Szegedy, *Spectral algorithms in higher-order Fourier analysis*, Forum of Mathematics, Sigma (2026), Vol. 14:e95 1–71, `doi:10.1017/fms.2026.10238 <https://doi.org/10.1017/fms.2026.10238>`_.
