Citing HoFa
======================================

If you would like to acknowledge the HoFa project in your academic publication, we suggest citing the following paper:

    Pablo Candela, Diego González-Sánchez, and Balázs Szegedy, *Spectral algorithms in higher-order Fourier analysis*, accepted for publication in Forum of Mathematics, Sigma, `doi:10.48550/arXiv.2501.12287 <https://doi.org/10.48550/arXiv.2501.12287>`_.
